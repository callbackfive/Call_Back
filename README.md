###### README
# CallBack 募資平台網站
CallBack是一個復刻的募資平台網站，參考國內外各大募資平台，以[FlyingV](https://www.flyingv.cc/)為復刻原型。
## 緣起
我們是[五倍紅寶石 Astro Camp](https://astro.5xruby.tw/)的第六屆學員。
這是我們在三個月內，從素人學習前後端技術，最終完成的團隊項目。

## 技術
### 前端
- Tailwind
- Javascript
- ReactJS
### 後端
- Ruby on Rails
### 資料庫
- PostgreSQL
### 部署
- Heroku (test CD)
